# etch-a-sketch

This project is for the Etch-a-Sketch challenge on The Odin Project. 

### Features

- [x] Square grid generation (default 16x16)
- [ ] Customizable settings
  - [x] Custom grid size (max 100x100)
  - [ ] Select drawing colour
  - [x] Rainbow drawing mode
- [x] Reset / clear grid
- [ ] Eraser
